The tabindex attribute

  • Elements with tabindex="0" are part of the natural tab sequence
  • Elements with tabindex="-1" are not, but are focusable with scripting
  • Elements with tabindex=">0" are never a good idea

The tabindex attribute (0)

<div>
    <button>1</button>
    <span tabindex="0" class="button">2</span>
    <button>3</button>
</div>

The tabindex attribute (0) demo

***

The tabindex attribute (-1)

<div>
    <button>1</button>
    <button tabindex="-1">2</button>
    <button>3</button>
</div>

The tabindex attribute (-1) demo

Accessibility APIs

Used by ATs to query accessibility information

Platform accessibility APIs

  • Windows: MSAA UIAutomation IAccessible2
  • Mac OS: NSAccessibility Protocol
  • Linux: IAccessible2 ATK/AT-ASPI
  • iOS: UIAccessibility
  • Android: Accessibility Framework

Accessibility Object Model (AOM)

JavaScript accessibility API for the future

Accessibility API mappings (AAM)

Define how role, state, properties, and keyboard focus are handled in the browser

  • Core Accessibility API Mappings (AAM) 1.1
  • HTML AMM 1.0
  • SVG AAM 1.0
